## Description
This application is a website that allows you to analyze specified web pages for SEO suitability.

---

## Used technologies
* Javalin
* JDBC
* H2 & PostgreSQL
* JUnit, AssertJ & MockWebServer
* JTE
* Lombok
* Bootstrap
* Docker (for deploy)
* And others
---

## Requirements
* JDK 20
* Gradle 8.2
---

## Usage
```bash
make install-dist
make start-dist
# Open http://localhost:8080
```
